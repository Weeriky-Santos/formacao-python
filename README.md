# 🐍 Formação Python: Do Básico a Aplicações de IA

Repositório criado para consolidar meu aprendizado em Python, conectando os fundamentos da linguagem, análise de dados e conceitos de Inteligência Artificial ao meu background em Business & Data Intelligence.

## 🎯 Objetivo
Aprofundar o domínio na linguagem Python para automatizar pipelines de dados, realizar análises avançadas com Pandas/NumPy e explorar integrações com IA, elevando o nível das entregas estratégicas de negócio.

---

## 📂 Estrutura do Repositório

* **`01-fundamentos/`**: Exercícios de lógica de programação, estruturas condicionais, manipulação de listas, dicionários, tuplas e funções.
* **`02-mini-projetos/`**: Projetos e desafios focados em análise exploratória com Pandas, NumPy, aplicações práticas e scripts completos desenvolvidos durante a formação.

## 🛠️ Tecnologias e Ferramentas
* Python 3.x
* Pandas & NumPy (Em desenvolvimento)
